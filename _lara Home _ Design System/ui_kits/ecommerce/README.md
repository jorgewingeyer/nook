# Élara Home — eCommerce UI Kit

High-fidelity interactive recreation of the Élara Home eCommerce storefront.

## Components

| File | Description |
|---|---|
| `Header.jsx` | Sticky nav with trust bar, logo, category nav, search, wishlist, cart |
| `Hero.jsx` | Full-viewport hero with headline, CTA, social proof stats |
| `ProductCard.jsx` | Product card with hover add-to-cart, wishlist toggle, badges |
| `CartDrawer.jsx` | Slide-in cart drawer with free shipping progress bar |
| `Footer.jsx` | Dark footer with newsletter signup, link columns, social icons |

## Usage

Open `index.html` in a browser. The prototype is fully interactive:
- Click "Explorá la colección" to scroll to the shop
- Toggle category filters
- Hover product cards to reveal "Agregar al carrito"
- Add products → cart drawer opens automatically
- Remove items from the cart drawer
- The free shipping progress bar updates dynamically

## Design Width
1440px desktop. Responsive to ~768px.
